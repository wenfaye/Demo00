一.云服务器常见问题：

问:服务器升级 ip 会变吗？数据会丢吗?对服务器有什么影响？

答:服务器升级后 ip 不会变化，数据不会丢失，不需要您重新部署环境或迁移数据，一般情况下升级CPU和内存需要重启云服务器，升级数据盘和带宽无需停止您的云服务器。

问:Linux 和 Windows 默认的系统盘大小是多少，是否可以自定义改变大小？

答:Linux 系统盘大小 20G，Windows 系统盘大小是 40G，大小固定，不能改变。

问:控制台重启服务器需要多长时间？

答:云服务器的重启操作一般需要 5-15 分钟之间，如果启动遇到文件系统自检检查扫描，也会影响重启的速度，若长时间未成功，请您提交售后支持工单寻求售后人员帮助。


二.云数据库 RDS 常见问题：

问:云数据库RDS for MySQL支持什么引擎？

答:支持 Myisam 和 Innodb 两种引擎，我们推荐您使用Innodb引擎，这将获得更好的性能（Myisam引擎虽不会产生错误，但由于Myisam引擎本身的缺限，可能带来表损坏场 景，所以我们建议用户使用Innodb引擎），不支持Memory引擎，如果您创建Memory引擎的表格，我们将自动为您转换成Myisam引擎的表 格。

问:应用程序是否需要支持自动重连数据库？

答:建议您的应用程序连接数据库的方式支持自动重连，这样可以提供更高的可用性。当数据库发生切换或迁移后，不需要您人工介入应用会自动恢复。同时建议您的应用程序采用长连接来连接数据库，短连接非常消耗性能和资源。


三.开放存储常见问题：

问:开放存储下载有峰值吗，如何收费？

备案用时多久送多久常见问题

问：一台服务器已经享受过“备案用时多久送多久”，用该服务器再次提交备案时，还能再次享受吗？

答：不能，同一台服务器只能享受一次赠送时长，不论您提交的网站是相同还是不同的网站，都不能再享受。


问：若备案过程中更换服务器，还能享受‘备案用时多久送多久’吗？

答：若备案过程中更换服务器，属于服务期限提前终止，不能享受赠送时长。


问：在备案成功前，服务器已到期，备案成功后还能享受‘备案用时多久送多久’吗？

答：服务器到期，您将无法享受赠送时长，请确保您的服务器在有效期内。


问：若您申请服务器退款，该服务器应享受的‘备案用时多久送多久’的赠送时长可以转移到其他服务器上吗？

答：若服务器发生退款，属于您提前终止合同，该服务器不能享受赠送时长，赠送时长也不能转移到其他服务器上。


问：若备案成功前，服务器做过升级或增加带宽的操作，赠送时长如何执行？

答：若备案成功前，您操作过升级或增加带宽，这样的操作不影响您的服务器时长，您的赠送时长将会被延长服务器期限后。
